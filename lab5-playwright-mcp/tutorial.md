# Lab 5 — Give Copilot a Browser (10 min)

## Overview

Give Copilot a **web browser** so it can open websites, click buttons, fill forms, and extract live information — all while chatting with you. No code needed — you'll plug in a ready-made MCP tool (Playwright) and watch the AI browse the web in real time.

## How to Start

Open Copilot Chat in **Agent** mode and type:

```
Start tutorial lab5
```

Copilot will guide you through everything step by step.
