# Lab 1 — Vibe-Code a Web App (10 min)

## Overview

Build a working web app **without writing any code** — just by describing what you want. You'll see the AI write code, run it, and update the app live in your browser. Then you'll iterate together, adding features one by one.

## How to Start

Open Copilot Chat in **Agent** mode and type:

```
Start tutorial lab1
```

Copilot will guide you through everything step by step.
