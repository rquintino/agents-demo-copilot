# GenAI & AI Agents — Session 3 Labs

## Welcome

Welcome to the hands-on labs for **Session 3: Agents, Tools & the ReAct Loop**.

You'll use **GitHub Copilot** — an AI coding assistant in VS Code — to explore how agents think, act, and extend their capabilities.

**Time:** ~55 minutes across 5 micro-labs

Click **Next** to set up your environment.

## Setup

Ensure you have VS Code with GitHub Copilot installed and authenticated.

No additional setup needed beyond that.

Click **Next** when ready.

## Lab 1 — Vibe-Code a Web App (10 min)

Navigate to the lab folder in VS Code:

Open the `lab1-web-app` folder and follow `tutorial.md` inside.

**Your mission:** Use Copilot to build and improve a web app — then preview it live.

**Debrief question:** *How did Copilot assist in the coding process? Could you trace the suggestions?*

## Lab 2 — Create Agent Skills (10 min)

Open the `lab2-skills` folder and follow `tutorial.md` inside.

**Your mission:** Teach Copilot a new skill — then watch it use it.

**Debrief question:** *How is a Skill different from a system prompt? When would you use each?*

## Lab 3 — Build an MCP Server (15 min)

Open the `lab3-mcp-server` folder and follow `tutorial.md` inside.

**Your mission:** Build a custom tool server that can be used with agents.

**Debrief question:** *What happens if the server fails? How should agents handle tool failures?*

## Lab 4 — AI Data Analysis (10 min)

Open the `lab4-data-analysis` folder and follow `tutorial.md` inside.

**Your mission:** Use Copilot to analyze data.

**Debrief question:** *How did Copilot handle errors in the analysis?*

## Lab 5 — Playwright MCP: Give Copilot a Browser (10 min)

Open the `lab5-playwright-mcp` folder and follow `tutorial.md` inside.

**Your mission:** Use a pre-built MCP server to give Copilot browser superpowers — navigate, click, screenshot, and extract data from live web pages.

**Debrief question:** *How does giving an agent a browser change what tasks you'd delegate to it? What guardrails would you add?*

## Wrap-Up

Congratulations! You've experienced the full agent stack:

| Lab | What you saw |
|-----|-------------|
| Lab 1 | Agent as **builder** — code generation + hot reload loop |
| Lab 2 | Agent with **skills** — packaged expertise, lazy-loaded |
| Lab 3 | Agent with **MCP tools** — extending capabilities via protocol |
| Lab 4 | Agent as **analyst** — error recovery, ReAct reasoning |
| Lab 5 | Agent with **browser** — pre-built MCP, web automation |

**The S3 punchline:** Same model for everyone. Your competitive edge = **context engineering** (S2) + **tools & skills** (S3).
