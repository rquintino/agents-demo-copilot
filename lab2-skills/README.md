# Lab 2 — AI Data Analysis

This lab contains sales data in `data/sales_data.csv`.
Copilot will create a Jupyter notebook (`analysis.ipynb`) and analyze the data step by step using Python (pandas, matplotlib).

The lab also demonstrates how the Sales Analyst skill in `.github/copilot-instructions.md` shapes Copilot's analysis style.

Use Python 3 with pandas, matplotlib, and jupyter.
If a library is missing, install it via `pip install -r requirements.txt`.
