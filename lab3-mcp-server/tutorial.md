# Lab 3 — Build an MCP Server (15 min)

## Overview

Give Copilot **new abilities** by building custom tools it can use. You'll explore a starter tool server, add new tools (like a company lookup), and then watch Copilot discover and use them live. Think of it as building plug-and-play accessories for an AI — using the MCP standard (the "USB for AI tools").

## How to Start

Open Copilot Chat in **Agent** mode and type:

```
Start tutorial lab3
```

Copilot will guide you through everything step by step.
