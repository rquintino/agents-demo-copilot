# Lab 3 — Build an MCP Server

This lab has a starter FastMCP server in server.py.
The student will extend it with new tools, then connect to use with Copilot or other agents.

To run the server: `python3 server.py`
To connect: Configure in VS Code or relevant agent settings.

The MCP server uses stdio transport by default (simplest for local development).
Use Copilot to help build and extend the server.
