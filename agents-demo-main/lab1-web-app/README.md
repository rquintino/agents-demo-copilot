# Lab 1 — Vibe-Code a Web App

This is a Vite + React project. When using Copilot to build or modify the app:
1. Edit files in src/
2. Run `pnpm dev` to start the dev server on port 5173
3. Preview via VS Code's port forwarding or browser

If pnpm is not available, install it or use npm.
Always use --host 0.0.0.0 when starting the dev server.
Keep the UI clean and modern. Use inline styles or a simple CSS approach.
Use Copilot to generate code suggestions and apply them.
